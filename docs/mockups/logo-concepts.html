Cogno Logo Explorations

Six possible identity directions for the game: all keep the dark arena mood, gold hierarchy, cyan memory energy, and a symbol that can shrink into a stage-select icon.

01 / Title LogoRecommended

Eye of Memory

Most aligned with the current title screen: instantly reads as memory, divinity, focus, and boss patterns.

02 / Seal MarkArcade Relic

Chronicle Seal

Feels like an ancient archive badge. Strong for app icons and menus, a bit more formal than playful.

03 / Memory CircuitHigh Energy

Neural Relic

Pushes the "cognition" side of Cogno. Better if we want history plus mind-game energy.

04 / Stage IdentityMenu Friendly

Boss Pantheon

Great for a stage-select game. It turns the historical bosses into the brand system from the start.

05 / Engraved TabletAncient

Engraved Tablet

More artifact-heavy and tactile. It gives the game weight, but can feel less sharp on small screens.

06 / Portal MarkCinematic

Memory Gate

Cinematic and readable on title screens. It is the most "enter the arena" direction.
