# Cogno - Next Steps

Última atualização: 2026-07-06

## Próximo passo imediato

Testar e validar o primeiro protótipo jogável no desktop e principalmente no iPhone.

## O que validar

- Se o drag direto está confortável no celular.
- Se o tamanho e a posição da arena funcionam bem sob o dedo.
- Se o player continua legível durante os ataques.
- Se os ataques vindos de fora da arena são compreensíveis.
- Se os telegraphs estão justos antes do dano.
- Se o modo real já parece uma fase curta com começo, meio e fim.
- Se o modo teste facilita avaliar cada ataque individualmente.
- Se Cleopatra já é visualmente reconhecível o suficiente para uma primeira versão.

## Depois da validação

Com base no teste, fazer melhorias de:

- responsividade mobile;
- conforto de controle;
- balanceamento de velocidade, duração e hitboxes;
- clareza dos telegraphs;
- impacto visual da Cleopatra e da arena;
- polimento dos quatro ataques iniciais;
- ajustes no script do modo real.
